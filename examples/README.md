# 使用例子

例子使用的`webpack-launcher`是本地源文件。

## 安装构建

```sh
git clone https://github.com/dog-days/webpack-launcher
cd webpack-launcher
npm install
```

## 运行

请保证在 clone 项目`webpack-launcher`根目录下运行命令。

### JavaScript 例子

原生 JavaScript 例子。

```sh
npm run javascript
```

### react 例子

react 例子，这个例子跟 create-react-app 一样。

```sh
npm run react
```

### custom 例子

customt 例子，自定义了 mock ，使用 less 等等自定义功能。

```sh
npm run react
```
